### ✨ المميزات

- كشف تلقائي للعملات السورية (قديمة وجديدة) باستخدام **YOLO11**
- دعم رفع الصور أو التقاطها مباشرة من كاميرا الموبايل
- حساب دقيق للمجموع (مع تحويل العملة الجديدة تلقائياً)
- إمكانية إضافة مبالغ يدوية (قديمة + جديدة)
- واجهة بسيطة وسريعة تعمل ممتاز على الموبايل
